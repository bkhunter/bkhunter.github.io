Blogface

January - April 2016

Imagine if you could make friends on Facebook with Google+, or any other kind of mainstream user. This is the primary goal of Blogface, a place to write, connect, and share.

Overview

The BlogFace is a social media style website made with Django, and was created as part of a group project for a web development course. This was an interesting project because groups were required to communicate with each other's services which necessitated the development of a RESTful api. My contribution to the project was mostly on the back end of the website which gave me first hand experience with the creation and testing of the API, as well as Django, PostgreSQL, and web hosting services. Interestingly, my group connected with the most groups throughout the class because our back end was one of the most stable. We believe this is because we ‘ate our own dog food’, meaning that our back end and front end were developed distinctly, which forced us to make calls to our own API as opposed to directly querying our database, and ensured the API functioned as required. I was very happy with this project, and I’m very grateful to have had the opportunity to build a dynamic and elastic website in a team setting.
